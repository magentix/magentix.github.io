Projets annexes

Leafiny est un CMS modulaire OpenSource simple et rapide. Il est conçu pour fonctionner sans dépendance et laisser le choix aux développeurs d'utiliser les techologies frontend souhaitées.

Le projet a débuté en 2014 avec une première version stable publique en novembre 2020.

  • Leafiny sur Github

StaPy est un mini générateur de site statique développé en Python. Il est axé sur la simplicité et l'interopérabilité : un seul fichier executable sur n'importe quel système disposant de Python, sans dépendance.

Le générateur permet de construire un mini site et de l'héberger gratuitement en quelques minutes sur une plateforme comme Netlify.

StaPy a également une vocation éducative. Un code simple, à modifier et enrichir selon les besoins.
